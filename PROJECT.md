# 项目背景

## 这是什么

家庭教育打印工具集。目标：**让家长选择内容 → 一键生成可打印的练习纸**，省去手写出题的时间。

目前包含三个子工具：
- **口算练习**（`kousuan.html`）— 1-6 年级数学口算
- **英语默写**（`english.html`）— 单词默写纸
- **精选套题**（`combo.html`）— 固定题型配比综合练习

⚠️ **注意：项目代码在 `project_20260601_092940/projects/` 下，不要在根目录下找。**

## 使用场景

- 家长在电脑上操作（选择年级、题型/单元、生成）
- 打印出来给孩子做
- 也支持"保存为 PDF"然后传到手机打印

## 技术约束

- **纯静态**：HTML + CSS + JS，无框架、无构建工具、无 npm
- **单文件优先**：每个工具一个 HTML 文件（内联 CSS/JS），数据可外置为 JS 文件
- **部署**：Coze 原生静态托管，开发时 `python -m http.server 5000`
- **浏览器兼容**：Chrome/Edge 打印效果优先
- **不依赖后端**：所有数据内嵌，localStorage 存偏好

## 设计令牌

两个工具共享同一套视觉语言（CSS 变量），差异仅在于 accent 色：
- 口算：暖橙 `#E8A317`
- 英语：蓝色 `#4A90D9`

纸张效果统一：米白背景 `#FDF5E8`，白色卡片 `#FFFFFF`，印刷墨色 `#3B3228`

## 教材版本

| 学科 | 教材 | 版本 |
|------|------|------|
| 口算 | 苏教版（通用） | 各年级大纲覆盖 |
| 英语 | 译林版（三起） | **2024 新版** |

英语教材 2024 年秋季启用了新版，单元主题和词汇与旧版完全不同。数据录入时必须确保是新版。
