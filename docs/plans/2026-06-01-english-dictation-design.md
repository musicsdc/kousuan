# 英语默写打印功能 — 设计文档

**日期**: 2026-06-01
**项目**: 口算练习生成器扩展
**版本**: v1.0

## 背景

现有口算练习生成器（`index.html`）支持 1-6 年级数学口算题生成和打印。用户希望新增英语单词默写功能，满足同样的"选择内容 → 一键生成 → 打印"需求。

## 教材版本

**译林版英语（2024 新版）**，三年级起点。2024 年秋启用的新教材，单元主题和词汇与旧版有显著差异。

覆盖范围：三年级上册 ~ 九年级下册（3A-9B，共 14 册）。**首期先做三年级上册（3A）跑通流程。**

## 文件结构

```
projects/
├── index.html              # 口算（不改动）
├── english.html            # 英语默写（新建）
├── js/
```

## 数据模型

```js
var wordBank = {
  "3A": {
    label: "三年级上册",
    units: [
      {
        unit: "Unit 1 Hello!",
        words: [
          { en: "hello", zh: "你好" },
          { en: "hi", zh: "嗨" }
        ]
      }
    ]
  },
  "3B": { ... },
  // 逐步扩充至 "9B"
};
```

- 仅存储核心词汇（不含 `*` 拓展词），与学校默写范围一致
- 短语/句子（如 "Good morning."）作为整体存储

## UI 设计

### 控制面板

与口算保持一致的卡片式控制面板，控件适配英语默写场景：

| 控件 | 说明 |
|------|------|
| 年级选择 | 3-9 年级分段按钮，默认 3 |
| 册别 | 上册 / 下册 切换 |
| 单元选择 | pill 按钮，多选，附全选/全不选快捷操作 |
| 模式切换 | 中→英（默认）/ 英→中 / 混合，segmented control |
| 每题数 | 10/20/30/自定义，默认 20 |
| 页数 | 1/2/5，默认 1 |
| 操作按钮 | 生成默写纸 / 打印·PDF |

### 打印纸

```
┌──────────────────────────────────────────┐
│          三年级上册 英语默写纸             │
│           Unit 1 Hello!                   │
│  姓名: ______  日期: ______  对题: ______  │
│                                          │
│  1. 你好          _______________        │
│  2. 猫            _______________        │
│  3. 同学们         _______________        │
│  ...                                     │
└──────────────────────────────────────────┘
```

- `cols-3` 网格布局（英文单词比口算式子长，3 列合适）
- 每题：序号 + 中文提示 + 下划线空格（中→英模式）
- 英→中模式：英文在前，中文空格在后
- 混合模式：随机穿插两种方向
- 多页时每页独立 paper 块，`@media print` 分页

### 单词选取逻辑

1. 用户勾选 N 个单元 → 合并所有单元词汇
2. 去重（同一 `en` 只保留一条）
3. 打乱顺序（shuffle）
4. 按每题数截取

## 公共样式策略

`english.html` 复用与 `index.html` 相同的设计令牌（CSS 变量 + 打印规则）。初期直接复制关键 CSS，后续可抽取 `print-common.css`。

## 实施策略

1. **Phase 1 — 3A 验证**：创建 `english.html` + 3A 单词数据，跑通完整流程
2. **Phase 2 — 全量数据**：补齐 3B-9B 单词表
3. **Phase 3 — 入口整合**：`index.html` 顶部添加 tab 切换（口算 / 英语默写）

## 数据来源

通过网络搜索"2024秋新译林版英语 X年级 单词表"，交叉验证多个教育站点后录入。每个年级标注数据来源和教材版本年份。
