# 家庭练习纸

> 给家长省去手写出题的小工具。纯静态，无需构建。

## 快速启动

```bash
python -m http.server 5000
# 浏览器打开 http://localhost:5000
```

## 页面一览

| 文件 | 用途 | 状态 |
|------|------|------|
| `index.html` | 首页（工具入口 + 使用说明） | ✅ |
| `kousuan.html` | 口算练习（1-6年级，25种题型） | ✅ |
| `english.html` | 英语默写（译林版 3-6年级，中英互默） | ✅ |
| `combo.html` | 精选套题（综合练习，50/100题） | ✅ |
| `js/wordbank.js` | 英语单词库（3A-6B，1065词条） | ✅ |
| `styles/common.css` | 共享CSS库 | ✅ |
| `assets/` | 教材图片、历史备份 | — |

## 当前状态（2026-06-13）

### 口算（kousuan.html）
- 1-6年级，加减乘除、分数、小数、方程、百分数、比例等
- A4打印优化，打印时自动加粗、分页、灰度

### 英语默写（english.html）
- 覆盖 3A-6B 共 8 个年级，64 单元，1065 词条
- 3-4 年级为 2024 新版译林版，5-6 年级为现行版
- 三种默写模式：中→英 / 英→中 / 混合
- 数据来源：教材配套单词表，OCR 提取 + 人工校对

### 精选套题（combo.html）
- 100题 / 50题双模式切换
- 题目+答案同页打印，裁切线分割

### 首页（index.html）
- 工具入口 + 使用说明 + 温馨提示，一页到底

## 已知问题

- 5-6 年级英语为旧版教材，需升级 2024 新版
- 4A 单元名为编号，需按课本补全标题
- 部分中文翻译由 OCR 提取，可能有误差
- `.btn-gen:hover` 阴影色各页需手动覆写
- 7-9 年级初中段未收录

## 更新规范

1. 改前先备份到 `assets/`
2. 修改后更新 `PROGRESS.md`
3. 单词数据严格按课本原文，只修 OCR 机械错误
